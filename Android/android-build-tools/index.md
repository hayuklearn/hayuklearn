# Android Build Tools

> SDK 管理器命令行工具

## sdkmanager

1. sdkmanager --list // 列出所有可用工具包
2. sdkmanager --uninstall "sdk-package-name" // 卸载
3. sdkmanager "sdk-package-name" // 安装
